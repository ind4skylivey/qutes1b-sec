# S1BGr0up - Quick Reference Card

## 🎯 MUST KNOW
| Action | Key/Command |
|--------|------------|
| Dashboard | `,h` or `home` |
| Search Dashboard | `Ctrl+K` |
| Add Node | Click `[+]` button |
| Fullscreen Terminal | Click `[⛶]` button |
| Toggle JS | `ts` |
| Terminal Help | `help` |
| New Tab | `t` |
| Close Tab | `x` |

---

## ⚡ NAVIGATION
```
j/k/h/l    Scroll (vim style)
gg / G     Top / Bottom
d / u      Page down/up
H / L      Back / Forward
r / R      Reload / Force reload
```

---

## 📑 TABS
```
t          New tab
x          Close
J / K      Prev / Next tab
gt / gT    Next / Prev
g0 / g$    First / Last
```

## 🔍 SEARCH & HINTS
